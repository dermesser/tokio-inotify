A stream which "fuse"s a stream once it's terminated.

Normally streams can behave unpredictably after they've terminated or returned an error, but Fuse is always defined to return None from poll after terination/errors, and afterwards all calls to schedule will be ignored.

fn is_done(&self) -> bool

Returns whether the underlying stream has finished or not.

If this method returns true, then all future calls to poll are guaranteed to return NotReady. If this returns false, then the underlying stream is still in use.
